# React + Vite

## ThreeJS 3D Developer Portfolio

Ce projet consiste à créer un portfolio de développeur impressionnant avec des graphiques et animations 3D en utilisant **ThreeJS** et **React Three Fiber**. Il fait partie d'un cours qui t'enseignera comment transformer ton site web en une expérience interactive captivante grâce à des modèles 3D et des animations fluides.

---

### Objectif du Projet

Apprendre à maîtriser **ThreeJS**, une bibliothèque puissante pour créer des graphismes 3D sur le web, ainsi que des outils modernes comme **React**, **TailwindCSS**, et **Framer Motion** pour créer une interface utilisateur dynamique et élégante. Ce projet est parfait pour ceux qui veulent explorer le développement web 3D et construire un portfolio professionnel qui se démarque.

---

### Technologies Utilisées

- **ThreeJS** : Bibliothèque pour le rendu et l'animation de modèles 3D.
- **React Three Fiber** : Intégration de ThreeJS avec React pour faciliter la création d'animations et d'objets 3D.
- **TailwindCSS** : Framework de styles CSS utilitaire-first pour un design rapide et responsive.
- **Framer Motion** : Librairie pour ajouter des animations fluides et interactives à ton site React.

---

### Fonctionnalités Principales

- **Chargement et Personnalisation de Modèles 3D** : 
  - Importe des modèles 3D personnalisés, ajuste la géométrie, applique différents types d'éclairages et place des objets dans un espace tridimensionnel.
  
- **Animations** : 
  - Apprends à animer les objets 3D avec fluidité en utilisant **ThreeJS** et **Framer Motion**.

- **Email Form** : 
  - Intégration d’un formulaire de contact pour envoyer des emails directement depuis le site web.

- **Optimisation des Performances et Responsiveness** : 
  - Utilisation des techniques comme **Suspense** et **Preload** pour rendre le site fluide et optimisé sur tous types d’appareils.

---

### Liens Utiles

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md) : Utilise **Babel** pour Fast Refresh.
